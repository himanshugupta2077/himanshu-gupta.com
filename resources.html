Learning Path

Pentesting Roadmap

A structured path from zero to professional pentester. Follow in order.

Phase 1 — Beginner
Build the Foundation
  • Networking basics (TCP/IP, DNS, HTTP)
  • Linux command line fluency
  • Python scripting basics
  • How the web works (requests, cookies, sessions)
  • Basic recon: nmap, gobuster, whatweb
  • Complete TryHackMe pre-security path
  • eJPT certification
  • OWASP Top 10 — read and understand each
  • PortSwigger Apprentice labs (all)
Phase 2 — Intermediate
Core Exploitation
  • Burp Suite — every feature, every extension
  • SQLi — manual exploitation, sqlmap mastery
  • XSS — reflected, stored, DOM, CSP bypass
  • IDOR, SSRF, XXE, SSTI deep dives
  • API security testing (REST, GraphQL)
  • JWT attacks and auth bypass
  • PortSwigger Practitioner labs (all)
  • HTB Starting Point + 10 machines
  • BSCP certification
  • Start bug bounty (HackerOne, Bugcrowd)
Phase 3 — Advanced
Professional Level
  • OSCP — pass the exam
  • Active Directory attacks
  • Internal network pentesting
  • Cloud security (AWS, Azure basics)
  • Custom exploit development
  • Code review for vulnerability research
  • eWPTXv2 certification
  • CVE research and responsible disclosure
  • Build your own tools and scripts
